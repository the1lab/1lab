# The nerve of a category {defines="nerve ordinal-category"}

Each object $[n]$ of the [[simplex category]] $\Delta$ presents an
honest category: the **ordinal category** generated by the [[poset of
finite ordinals|finite-ordinal-poset]] $0 \le 1 \le \dots \le n$. Since
monotone maps induce functors, every $\Delta$-map gives a functor
between ordinal categories.

```agda
ordinal : Nat → Precategory lzero lzero
ordinal n = poset→category (Fin-poset (suc n))

Δ-map→functor : ∀ {n m} → Δ-map n m → Functor (ordinal n) (ordinal m)
Δ-map→functor f = monotone→functor λ where
  .hom          → f .map
  .pres-≤ {x} {y} p → f .ascending x y p
```

The **nerve** of a [[strict category]] $\cC$ is the [[simplicial
set|simplicial-set]] whose $n$-simplices are the functors $[n] \to
\cC$: vertices are objects, edges are morphisms, and an $n$-simplex is
a string of $n$ composable morphisms together with all their
composites. Strictness is needed so that these functors form a *set*.
Restriction along a $\Delta$-map is precomposition, which is
definitionally functorial; the only thing to check is that the two
ways of writing down each functor agree on the proof fields, which is
exactly what `Functor-path`{.Agda} does.

```agda
nerve : (C : Precategory lzero lzero) → is-set ⌞ C ⌟ → ⌞ sSet ⌟
nerve C ob-set .F₀ n = el (Functor (ordinal n) C) (Functor-is-set ob-set)
nerve C ob-set .F₁ f F = F F∘ Δ-map→functor f
nerve C ob-set .F-id = funext λ F →
  Functor-path (λ _ → refl) (λ _ → refl)
nerve C ob-set .F-∘ f g = funext λ F →
  Functor-path (λ _ → refl) (λ _ → refl)
```

## The delooping groupoid of gauge theory {defines="simplicial-delooping"}

For a [[monoid]] $M$ — in gauge-theoretic applications, a group $G$ of
gauge transformations — the nerve of the [[delooping category]]
$\B{M}$ is the simplicial incarnation of the delooping: it has a
single vertex, one edge for every element of $M$, and one 2-simplex
for every *pair* of elements, witnessing their composition.

In the notation of Schreiber's *Higher topos theory in physics*, the
plots of shape $\Delta^n$ of this simplicial set compute as
$\rm{Plt}(\Delta^0, \B{M}) \simeq \{\rm{pt}\}$,
$\rm{Plt}(\Delta^1, \B{M}) \simeq M$ and
$\rm{Plt}(\Delta^2, \B{M}) \simeq M \times M$. We start with the
vertices: a functor out of $[0]$ has nowhere to go but the point.

```agda
  nerve-B₀-is-contr : is-contr ⌞ nerve-B .F₀ 0 ⌟
  nerve-B₀-is-contr .centre .F₀ _ = tt
  nerve-B₀-is-contr .centre .F₁ _ = mon.identity
  nerve-B₀-is-contr .centre .F-id = refl
  nerve-B₀-is-contr .centre .F-∘ _ _ = sym mon.idl
  nerve-B₀-is-contr .paths G = Functor-path (λ _ → refl) λ {x} {y} p → go x y p
    where
    go : ∀ x y (p : x ≤ y) → mon.identity ≡ G .F₁ p
    go (fin 0) (fin 0) p = sym (ap (G .F₁) (Nat.≤-is-prop p _) ∙ G .F-id)
    go (fin (suc k) ⦃ b ⦄) y p = absurd (Nat.¬suc≤0 (Nat.≤-peel b))
    go x (fin (suc k) ⦃ b ⦄) p = absurd (Nat.¬suc≤0 (Nat.≤-peel b))
```

An edge is determined by exactly one monoid element: the image of the
generating arrow $0 \le 1$.

```agda
  nerve-B₁≃M : ⌞ nerve-B .F₀ 1 ⌟ ≃ M
  nerve-B₁≃M = Iso→Equiv (to , iso from (λ _ → refl) from-to) where
    to : Functor (ordinal 1) BM → M
    to F = F .F₁ {fzero} {fsuc fzero} Nat.0≤x

    edge : M → Fin 2 → Fin 2 → M
    edge g (fin 0) (fin 1) = g
    edge g _ _ = mon.identity

    edge-id : ∀ g (i : Fin 2) → edge g i i ≡ mon.identity
    edge-id g (fin 0) = refl
    edge-id g (fin 1) = refl
    edge-id g (fin (suc (suc k)) ⦃ b ⦄) =
      absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel b)))

    edge-∘
      : ∀ g (x y z : Fin 2) → y ≤ z → x ≤ y
      → edge g x z ≡ edge g y z mon.⋆ edge g x y
    edge-∘ g (fin 0) (fin 0) (fin 0) p q = sym mon.idl
    edge-∘ g (fin 0) (fin 0) (fin 1) p q = sym mon.idr
    edge-∘ g (fin 0) (fin 1) (fin 1) p q = sym mon.idl
    edge-∘ g (fin 1) (fin 1) (fin 1) p q = sym mon.idl
    edge-∘ g (fin 0) (fin 1) (fin 0) p q = absurd (Nat.¬suc≤0 p)
    edge-∘ g (fin 1) (fin 0) z p q = absurd (Nat.¬suc≤0 q)
    edge-∘ g (fin (suc (suc k)) ⦃ b ⦄) y z p q =
      absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel b)))
    edge-∘ g x (fin (suc (suc k)) ⦃ b ⦄) z p q =
      absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel b)))
    edge-∘ g x y (fin (suc (suc k)) ⦃ b ⦄) p q =
      absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel b)))

    from : M → Functor (ordinal 1) BM
    from g .F₀ _ = tt
    from g .F₁ {x} {y} _ = edge g x y
    from g .F-id {x} = edge-id g x
    from g .F-∘ {x} {y} {z} p q = edge-∘ g x y z p q

    from-to : ∀ F → from (to F) ≡ F
    from-to F = Functor-path (λ _ → refl) λ {x} {y} p → go x y p where
      go : ∀ x y (p : x ≤ y) → edge (to F) x y ≡ F .F₁ p
      go (fin 0) (fin 0) p = sym (ap (F .F₁) (Nat.≤-is-prop p _) ∙ F .F-id)
      go (fin 0) (fin 1) p = ap (F .F₁) (Nat.≤-is-prop _ p)
      go (fin 1) (fin 1) p = sym (ap (F .F₁) (Nat.≤-is-prop p _) ∙ F .F-id)
      go (fin 1) (fin 0) p = absurd (Nat.¬suc≤0 p)
      go (fin (suc (suc k)) ⦃ b ⦄) y p =
        absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel b)))
      go x (fin (suc (suc k)) ⦃ b ⦄) p =
        absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel b)))
```

And, as in Schreiber's (27), a 2-simplex of $\B{M}$ is exactly a pair
of monoid elements $(g_1, g_2)$ — the third edge is forced to be the
composite $g_2 \cdot g_1$, and the 2-simplex is the witness of this
composition.

```agda
  nerve-B₂≃M×M : ⌞ nerve-B .F₀ 2 ⌟ ≃ (M × M)
  nerve-B₂≃M×M = Iso→Equiv (to , iso from (λ _ → refl) from-to) where
    to : Functor (ordinal 2) BM → M × M
    to F = F .F₁ {i0'} {i1'} Nat.0≤x , F .F₁ {i1'} {i2'} (s≤s Nat.0≤x)

    edge : M × M → Fin 3 → Fin 3 → M
    edge (g , h) (fin 0) (fin 1) = g
    edge (g , h) (fin 1) (fin 2) = h
    edge (g , h) (fin 0) (fin 2) = h mon.⋆ g
    edge (g , h) _ _ = mon.identity

    edge-id : ∀ gs (i : Fin 3) → edge gs i i ≡ mon.identity
    edge-id gs (fin 0) = refl
    edge-id gs (fin 1) = refl
    edge-id gs (fin 2) = refl
    edge-id gs (fin (suc (suc (suc k))) ⦃ b ⦄) =
      absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel (Nat.≤-peel b))))

    edge-∘
      : ∀ gs (x y z : Fin 3) → y ≤ z → x ≤ y
      → edge gs x z ≡ edge gs y z mon.⋆ edge gs x y
    edge-∘ gs (fin 0) (fin 0) (fin 0) p q = sym mon.idl
    edge-∘ gs (fin 0) (fin 0) (fin 1) p q = sym mon.idr
    edge-∘ gs (fin 0) (fin 0) (fin 2) p q = sym mon.idr
    edge-∘ gs (fin 0) (fin 1) (fin 1) p q = sym mon.idl
    edge-∘ gs (fin 0) (fin 1) (fin 2) p q = refl
    edge-∘ gs (fin 0) (fin 2) (fin 2) p q = sym mon.idl
    edge-∘ gs (fin 1) (fin 1) (fin 1) p q = sym mon.idl
    edge-∘ gs (fin 1) (fin 1) (fin 2) p q = sym mon.idr
    edge-∘ gs (fin 1) (fin 2) (fin 2) p q = sym mon.idl
    edge-∘ gs (fin 2) (fin 2) (fin 2) p q = sym mon.idl
    edge-∘ gs (fin 1) (fin 0) z p q = absurd (Nat.¬suc≤0 q)
    edge-∘ gs (fin 2) (fin 0) z p q = absurd (Nat.¬suc≤0 q)
    edge-∘ gs (fin 2) (fin 1) z p q = absurd (Nat.¬suc≤0 (Nat.≤-peel q))
    edge-∘ gs (fin 0) (fin 1) (fin 0) p q = absurd (Nat.¬suc≤0 p)
    edge-∘ gs (fin 0) (fin 2) (fin 0) p q = absurd (Nat.¬suc≤0 p)
    edge-∘ gs (fin 0) (fin 2) (fin 1) p q = absurd (Nat.¬suc≤0 (Nat.≤-peel p))
    edge-∘ gs (fin 1) (fin 1) (fin 0) p q = absurd (Nat.¬suc≤0 p)
    edge-∘ gs (fin 1) (fin 2) (fin 0) p q = absurd (Nat.¬suc≤0 p)
    edge-∘ gs (fin 1) (fin 2) (fin 1) p q = absurd (Nat.¬suc≤0 (Nat.≤-peel p))
    edge-∘ gs (fin 2) (fin 2) (fin 0) p q = absurd (Nat.¬suc≤0 p)
    edge-∘ gs (fin 2) (fin 2) (fin 1) p q = absurd (Nat.¬suc≤0 (Nat.≤-peel p))
    edge-∘ gs (fin (suc (suc (suc k))) ⦃ b ⦄) y z p q =
      absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel (Nat.≤-peel b))))
    edge-∘ gs x (fin (suc (suc (suc k))) ⦃ b ⦄) z p q =
      absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel (Nat.≤-peel b))))
    edge-∘ gs x y (fin (suc (suc (suc k))) ⦃ b ⦄) p q =
      absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel (Nat.≤-peel b))))

    from : M × M → Functor (ordinal 2) BM
    from gs .F₀ _ = tt
    from gs .F₁ {x} {y} _ = edge gs x y
    from gs .F-id {x} = edge-id gs x
    from gs .F-∘ {x} {y} {z} p q = edge-∘ gs x y z p q

    from-to : ∀ F → from (to F) ≡ F
    from-to F = Functor-path (λ _ → refl) λ {x} {y} p → go x y p where
      go : ∀ x y (p : x ≤ y) → edge (to F) x y ≡ F .F₁ p
      go (fin 0) (fin 0) p = sym (ap (F .F₁) (Nat.≤-is-prop p _) ∙ F .F-id)
      go (fin 1) (fin 1) p = sym (ap (F .F₁) (Nat.≤-is-prop p _) ∙ F .F-id)
      go (fin 2) (fin 2) p = sym (ap (F .F₁) (Nat.≤-is-prop p _) ∙ F .F-id)
      go (fin 0) (fin 1) p = ap (F .F₁) (Nat.≤-is-prop _ p)
      go (fin 1) (fin 2) p = ap (F .F₁) (Nat.≤-is-prop _ p)
      go (fin 0) (fin 2) p =
        sym (F .F-∘ _ _) ∙ ap (F .F₁) (Nat.≤-is-prop _ p)
      go (fin 1) (fin 0) p = absurd (Nat.¬suc≤0 p)
      go (fin 2) (fin 0) p = absurd (Nat.¬suc≤0 p)
      go (fin 2) (fin 1) p = absurd (Nat.¬suc≤0 (Nat.≤-peel p))
      go (fin (suc (suc (suc k))) ⦃ b ⦄) y p =
        absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel (Nat.≤-peel b))))
      go x (fin (suc (suc (suc k))) ⦃ b ⦄) p =
        absurd (Nat.¬suc≤0 (Nat.≤-peel (Nat.≤-peel (Nat.≤-peel b))))
```

For a *group*, every horn of this simplicial set has a filler — the
delooping is a [[Kan complex]], the delooping *groupoid* $\B{G}$ of
gauge theory — but we leave the general filling argument for future
work.
