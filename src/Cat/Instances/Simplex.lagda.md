# The simplex category {defines="simplex-category"}

The simplex category, $\Delta$, is generally introduced as the category
of non-empty finite ordinals and order-preserving maps. We will, for
convenience reasons, define it _skeletally_: Rather than taking actual
finite ordinals as objects, we will use the natural numbers as ordinals,
each natural standing for the isomorphism class of ordinals of that
cardinality.

```agda
record Δ-map (n m : Nat) : Type where
  field
    map       : Fin (suc n) → Fin (suc m)
    ascending : (x y : Fin (suc n)) → x ≤ y → map x ≤ map y
```

```agda
Δ : Precategory lzero lzero
Δ .Ob = Nat
Δ .Hom n m = Δ-map n m
Δ .Hom-set _ _ = hlevel 2

Δ .id .map x = x
Δ .id .ascending x y p = p

Δ ._∘_ f g .map x = f .map (g .map x)
Δ ._∘_ f g .ascending x y p = f .ascending _ _ (g .ascending _ _ p)

Δ .idr f = Δ-map-path λ x → refl
Δ .idl f = Δ-map-path λ x → refl
```

## Faces and degeneracies {defines="coface codegeneracy"}

The morphisms of $\Delta$ are generated by two families of maps
determined by `skip`{.Agda} and `squish`{.Agda}: the **coface** maps
$\delta_i : [n] \to [n+1]$, the injections which miss the $i$-th
vertex, and the **codegeneracy** maps $\sigma_j : [n+1] \to [n]$, the
surjections which hit the $j$-th vertex twice.

```agda
δ : ∀ {n} → Fin (suc (suc n)) → Δ-map n (suc n)
δ i .map = skip i
δ i .ascending = skip-monotone i

σ : ∀ {n} → Fin (suc n) → Δ-map (suc n) n
σ i .map = squish i
σ i .ascending = squish-monotone i
```

The *simplicial identities*, proved for `skip`{.Agda} and
`squish`{.Agda} in [`Data.Fin.Properties`], lift to equations between
these generators, which describe all the ways a face or degeneracy can
be moved past another.

[`Data.Fin.Properties`]: Data.Fin.Properties.html

```agda
δ-comm
  : ∀ {n} (i j : Fin (suc (suc n))) → i ≤ j
  → δ (weaken i) ∘Δ δ j ≡ δ (fsuc j) ∘Δ δ i
δ-comm i j le = Δ-map-path (skip-comm i j le)

σ-comm
  : ∀ {n} (i j : Fin (suc n)) → i ≤ j
  → σ j ∘Δ σ (weaken i) ≡ σ i ∘Δ σ (fsuc j)
σ-comm i j le = Δ-map-path (drop-comm i j le)

δ-σ-comm
  : ∀ {n} (i : Fin (suc (suc n))) (j : Fin (suc n)) → i < fsuc j
  → σ (fsuc j) ∘Δ δ (weaken i) ≡ δ i ∘Δ σ j
δ-σ-comm i j lt = Δ-map-path (squish-skip-comm i j lt)

σ-δ-id
  : ∀ {n} (j : Fin (suc n))
  → σ j ∘Δ δ (weaken j) ≡ Δ .id
σ-δ-id j = Δ-map-path (squish-skip j j refl)

σ-δ-id'
  : ∀ {n} (j : Fin (suc n))
  → σ j ∘Δ δ (fsuc j) ≡ Δ .id
σ-δ-id' j = Δ-map-path (squish-skip-fsuc (fsuc j) j refl)
```
